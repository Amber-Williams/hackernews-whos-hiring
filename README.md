# hackernews-whos-hiring

Parses the hackernew's who's hiring thread into an easy to read markdown table


## Running the repo
### Prerequisites
- Python 3.10+
- poetry
- copy .env.example to .env and add your API keys
### Steps

```bash
poetry install
poetry run python main.py
```
